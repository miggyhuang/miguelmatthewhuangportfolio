This page contains my reflections on the topics and what I understood:



During the last quarter of classes, I have introduced to even more uses and functions of Javascript. We discussed mainly 4 topics this quarter which teaches us the basics of making a dynamic websites by implementing these different events, event handlers, pop-up boxes, conditional statements, and loops. So let us make an overview of these 4 topics. Firstly, events are anything that can trigger a function. It can be a mouse click, loading of an image or even pressing a key on your keyboard. Pop-up boxes are alerts, reminders, user-input or confirmation objects that appear when we cause an event connected to the functions show_alert, show_prompt or show_confirm. When we want to put different results for different inputs we can use conditional statements. With this we can put what outputs will appear when this type of input is entered. Examples include if, if-else, if-else-if, and switch. The return statement is for when we want to have an output based on the operations to be executed. Loops serve the purpose of allowing repetative blocks of codes to be executed until a condition is met. There are different types of loops we can use like the for(specific no. of iterations), while(continues until condiiton met), do-while(just like while, but executes code atleast once) and for-in(loops elements through an array), and we can also use the break(stops altogether) and continue(stops and proceeds with the next set value) statements to stop a loop.
