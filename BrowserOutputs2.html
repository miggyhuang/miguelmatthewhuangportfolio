Lesson 1:Introduction to CSS:

CSS stands for Cascading Markup Language and is used as the presentation layer of a webpage. With the help of CSS, we can customize the background, background color, size and fonts of texts, and the height and width of images. Basically how elements will be displayed on the screen.

Lesson 2:Types of CSS

There are 3 types of CSS. Inline which uses the attribute styles to style a single line element, an embedded, internal or header CSS which uses the style tag in betweemn the head tags and is used to apply more complex css properties and finally linked or external CSS, where the user will type in the syntax in a different text code which is linked to the main HTML page via the link tag

Lesson 3:Selectors

Selectors can be used for applying a single CSS syntax to multiple specific elements without keeping the code too long so that those elements affected look identitcal to one another. The class selector is used for multiple elements while the id is a unique syntax given to only 1 element. The universal is used to apply a syntax to all elements in the webpage, and the attribute selector is for selecting elements with a specific attirbute value.

Lesson 4: Display, Div and Span, Class and ID Selector

Div tags are used for big chunks of texts and paragraphs. It can be used for applying styles to different sections of the page. Span tags are used for smaller chunks of texts and should not be used for multiple paragraphs. It can be used to apply to an inline element and empahsize words or sentences. Class selectors are written using the format (selector).(class name) or .(class name) and ids are declared with #(id name)

Lesson 5:Pseudo-classes

We can use pseudo-classes and its properties to style a link when it is in its unvisited, visited, hovered or active state. a:link is for styling unvisited links while a:visited is for customizing visited links. The a:hover can be used when we want to apply an effect that occurs when a mouse cursor hovers over the element. "a:active" is used if the user wants to apply a style to the link when it is pressed.

Lesson 6:Formatting Text with CSS

There are many ways we can format texts in CSS. These include font-weight, which is what we use to apply a bold or lighter effect, the font-family, to determine the font to be used, font-size, where we can style the size of the text, font-style, the property used for italicizing, color, which we can use to give the selected text a color, text-align, so that we can align the text in a specific way, and word-spacing, which is used to determine the space in between words. There are much more options we can use but these are the most used ones.

Lesson 7:Formatting Layout with CSS

The box model is a concept we can use to understand how the space between elements work. We can also contorl these with the help of CSS. The height, width, margin, padding, etc are collectivley known as the dimensions of an element. We can also set the position of an element with the help of the position property. Float property can be used to indicate where the next elements can be displayed while the background-repeat of an element can also be controlled. An example of another dimension we can control is the margin properties which as the property states, controlls the margin of an element.
