Pseudo-classes

aespa - Drama MV

When this link is not clicked and just displayed it shows the color #00ffff with the help of the a:link.

aespa - Armageddon MV

This link will display the color #ffe4c4 if visited by the user. To do this we use the a:visited pseudo class.

aespa - Supernova MV

When we hover the mouse over this link, we can see that it changes its color from #00ffff to #a52a2a because oft he a:hover.

aespa - Whiplash MV

This click when actively clicked has the color styled as #556b2f. This can be achieved by the use of the pseudo-class a:active.
