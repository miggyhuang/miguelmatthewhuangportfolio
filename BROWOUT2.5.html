Pseudo-classes

aespa - Drama MV

When this link is not clicked and just displayed it shows the color #00ffff with the help of the a:link.

aespa - Armageddon MV

This link will display the color #ffe4c4 if visited by the user. To do this we use the a:visited pseudo class.

aespa - Supernova MV

When we hover the mouse over this link, we can see that it changes its color from #00ffff to #a52a2a because oft he a:hover.

aespa - Whiplash MV

This click when actively clicked has the color styled as #556b2f. This can be achieved by the use of the pseudo-class a:active.

We can use pseudo-classes and its properties to style a link when it is in its unvisited, visited, hovered or active state. a:link is for styling unvisited links while a:visited is for customizing visited links. The a:hover can be used when we want to apply an effect that occurs when a mouse cursor hovers over the element. "a:active" is used if the user wants to apply a style to the link when it is pressed.
