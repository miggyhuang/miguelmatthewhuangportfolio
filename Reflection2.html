My Reflection

I learned a lot of new things in ICT this quarter, and these are what I now know and will always keep in mind when creating a webpage:

What I learned in ICT?

As a student, learning how to use CSS and the capabilites of the language helped me in the design and presentation aspect of my webpages. Thanks to it, I was able to have more control over the customization and style of the text, videos and even images. CSS is what helps give the looks and is in charge of making the website presentable. CSS can be used in 3 ways. Inline to define a single elemennt, embedded which are placed in the head tag, and linked CSS, a way of writing CSS code in a whole different page yet still affects the HTML page it is linked to. Selectors are a way of assigning specific CSS syntaxes to specific elements. Among these are the class, universal, id and attribute selectors. The ones that are most used are the class and id selectors. The class selector is best used when assigning a syntax to multiple elements so that they all have identical formats. It is written with either the element followed by a period (.) then assigning it a class name, or period (.) then the class name written after. IDs on the other hand should be used when giving only a single line a specific CSS syntax. To declare one, we use the hashtag (#) then assign it a unique ID name. Div and span are used if you want to give text a specific word format. You can use div for big chunks of text and span for small ones. To apply the CSS syntax, you may use the class or id selectors. Links can also have CSS syntaxes applied on them. The a:link is used for setting effects on an unvisited link, while a the a:visited is for those links that were visited. a:hover can be used to give characteristics to a link that actively has a mouse hovering over it. Lastly a:active can be used if you want to give an effect to a link that is actively being clicked. Collectively these are what we call pseudo-classes. With the help of CSS, we can also format texts to give it a unique effect. Some of these include text-align (align text on a webpage), text-decoration (indicating the decoration applied), font-size (for controlling the size), font-family (giving text a specific font) or even word spacing (to declare the amount fo space between words). This can be crucially helpful in giving a website emphasis on some phrases and a sense of cohesion. The box model is a concept that each element on the webpage has its own box that borders the boxes of other elements. CSS gives us control over that box and how we can present those elements. This is called the dimensions which can be modified with the helf of width and height tags. The background image of our webpage is also customizable. If the picture is not big enough to cover the space, we can set a background repeat so that the image covers the whole screen. At the same time, we can also use no-repeat to disable it if the page looks unpresentable with it.
